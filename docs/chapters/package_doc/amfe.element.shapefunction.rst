amfe.element.shapefunction module
=================================

The amfe.element.shapefunction module implements Shapefunctions.
Currently the following shapefunctions are available:

- Line2
- Line3
- Tri3
- Tri6
- Quad4
- Quad8
- Hexa8
- Hexa20
- Tet4
- Tet10

The base module provides base classes for implementations of such
shapefunctions.
